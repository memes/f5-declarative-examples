# Declarative Examples for F5 solutions on Google Cloud

![GitHub release](https://img.shields.io/github/v/release/memes/f5-declarative-examples?sort=semver)
![Maintenance](https://img.shields.io/maintenance/yes/2025)
[![Contributor Covenant](https://img.shields.io/badge/Contributor%20Covenant-2.1-4baaaa.svg)](CODE_OF_CONDUCT.md)

This repo aims to be a collection of tested onboarding and configuration declarations to integrate F5 products in Google
Cloud. The motivation for this is to have a centralized collection that F5 sales teams and customers can use as basis
for their own use-cases independent of the tools used to provision Google Cloud resources.
